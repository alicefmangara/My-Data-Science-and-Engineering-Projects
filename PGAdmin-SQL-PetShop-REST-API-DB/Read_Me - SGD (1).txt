Read_Me - SGD
Intrucoes para correr o projeto sgd_2023_api.py

Antes de começar, é necessário passar os ficheiros "sgd_2023_api" para a extensão python (.py) e "requests" para .json

Posteriormente é necssário criar um virtual environment em django e importar as necessárias packages seguindo as instrucoes:

Começar por mudar o diretório para a pasta onde tal projeto se encontra

$ python3 -m venv django_env
$ source django_env/bin/activate
$ pip3 install flask
$ pip3 install psycopg2-binary
$ python3 sgd_2023_api.py

Após seguir as instruções acima, introduzir o ficheiro .json no Postman de forma a testar todas as funcionalidades desenvolvidas.
